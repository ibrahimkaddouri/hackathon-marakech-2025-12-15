# 100% HR Hackathon

A 6-hour hackathon competition where teams build multimodal HR agents.

## About

Teams compete to create innovative HR agents that accept input via different modalities:
- **Voice** - Voice-based interaction
- **Text** - Text-based chat interfaces  
- **Image** - Visual/document processing

Each team specializes in one modality and builds their unique solution.

## Team Roles

Each team consists of 4 specialists:
- **Product** - Product strategy and design
- **Marketing** - Go-to-market and presentation
- **Software** - Development and implementation
- **AI** - AI/ML integration and optimization
